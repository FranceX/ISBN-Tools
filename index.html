Isbn-tools

Formats an ISBN-10 or ISBN-13 with dashes. Validates ISBN-10 or ISBN-13.

View on GitHub Download .zip Download .tar.gz

isbn_tools.php:

Takes an unformatted ISBN-10 or ISBN-13, and formats it with dashes using the RangesMessage.xml file from http://www.isbn-international.org/export_rangemessage.xml

Also provides a validating function.

Usage: formatISBN($ISBN) where $ISBN is an unformatted ISBN-10 or ISBN-13 returns a formatted ISBN-10 or ISBN-13 validateISBN($ISBN) where $ISBN is an unformatted ISBN-10 or ISBN-13 returns true if valid, false if not.

example.php

To test, place example.php on a server with PHP and open it in a browser.

Enter some ISBNs without dashes like 9788992589109 9789868379206 and click Submit to see them formatted and validated
